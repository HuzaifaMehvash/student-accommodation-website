
# Student Accommodation Website

A responsive PG Finder web application built using PHP, MySQL, Bootstrap, JavaScript, AJAX, and React.

## Features
- Property Listing
- Property Details
- Login/Signup
- Interested Properties
- AJAX Filtering
- Responsive Design

## Technologies Used
- HTML
- CSS
- Bootstrap
- PHP
- MySQL
- AJAX
- JavaScript
